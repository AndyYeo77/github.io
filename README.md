# Explore Voice Agent

A simple browser-based voice assistant built with the Web Speech API.

## Features

- Start and stop voice listening with a button
- Transcribes spoken text in real time
- Produces spoken agent responses with speech synthesis
- Handles basic exploration commands like:
  - "Tell me about Mars"
  - "Search for nearby parks"
  - "What can you do?"

## Usage

1. Open `index.html` in a modern browser like Chrome or Edge.
2. Allow microphone access when prompted.
3. Click **Start Listening** and speak a command.

## Files

- `index.html` — app UI
- `style.css` — app styling
- `script.js` — voice recognition and response logic
